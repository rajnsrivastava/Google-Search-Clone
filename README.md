# Google-Search-Clone
🔍 Google Search Clone
A fully functional web application that replicates the look and feel of the Google Search engine.

🚀 Live Demo

✨ Features
Real-Time Search: Fetches live data from the web using the Google Custom Search API.

Image Search: Dedicated tab to browse image results specifically.

Clean UI: A pixel-perfect recreation of the Google homepage and results page.

Responsive Design: Optimized for mobile, tablet, and desktop views.

Fast Performance: Built with React for a smooth, single-page application (SPA) experience.



🛠️ Technologies Used
Frontend: HTML, CSS
Deployment: Netlify
